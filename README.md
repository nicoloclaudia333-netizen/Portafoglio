# Portfolio - Claudia Nicolò

Sito portfolio personale realizzato come progetto finale del Master UX/UI Design di Start2impact University.

## Pagine

- **Home** — presentazione e bio
- **CV** — curriculum vitae in HTML con download PDF
- **Progetti** — galleria dei lavori realizzati durante il corso
- **Contatti** — form di contatto collegato a Formspree

## Tecnologie utilizzate

- HTML5
- Sass/SCSS compilato in CSS3
- Bootstrap 5
- Font Awesome 6
- Google Fonts (Poppins, Dancing Script)
- Formspree (invio form)

## Funzionalità

- Layout responsive (mobile, tablet, desktop)
- Menu sticky su mobile
- Flexbox e CSS Grid per i layout
- Favicon personalizzata
- Tag Open Graph per la condivisione sui social
- Animazioni CSS
